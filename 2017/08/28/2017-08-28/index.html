流水账：2017-08-28

发表于 2017-08-28 | 更新于 2017-08-28 | 分类于 个人日记 | | 共 1,096 字 | 阅读 次

还是喜欢晚上，安静，不会被打扰。

关于本博客

其实，在家这段时间，情绪几度起伏。太闲导致无所事事，胡思乱想。但是又不想写在微博上，总感觉会给人一种无病呻吟的感觉。但是，我又有一些情感或者心情无法表达，导致我心情很是苦闷。所以，当我无法排解的时候，我会记录在我的博客上。我想看到的也没几个人。哈哈哈~ 但是终究会有人会看到，所以，看到的读者，假如不喜欢希望不要吐槽我，哈哈哈~

我也是一个有些有肉的人，我有我的经历、心思、想法。我有我无法释怀的经历🙏，我也想早日走出阴影，我也在努力改变，努力学习，努力充实自己，锻炼自己。也希望过去的时候能够有个算是圆满的结局。

所以，在本博客中，会出现很多我的个人情感。在一定程度上讲，我也很想多写一些关于技术的东西，而不是单纯的个人情感方面的东西，这些东西犹如一个长不大孩子的日记。谁不想成熟一点，写出的东西高大上一些呢。哪怕是一些工作生活遇到的问题的思考也是有意义的。

而今，我经历着我暂时不太容易自我解决的心理问题，我需要时间慢慢疗伤，我也就想稍稍的放纵一下自己，去展露一些我的内心。没有多少人会看到，但是，我总需要写出来权当发泄一下吧！🙏我是为了我自己。

关于最近

在经历了我的第三次科目二考试之后，我终于通过了考试，可以准备科目四的考试了。预计将在本月三十一号下午进行科目四考试。届时只要通过考试只需要等最多三天驾驶证就可以寄到家了。所以，我暂定九月三号离家。

回家已经134天了。

最近也一直在做去深圳的准备，联系之前公司的同事，在豆瓣上看了看各种租房信息。前几天改了简历，一直在看招聘启事。就我现在看到的情况，我感觉深圳机会是多了，但是，竞争也变激烈了。我还是要做好心理准备的。

这几天都在看科目四的试题，所以，也没怎么去看关于技术的东西。前些天，大约二十号之前，我投了济南的一家做电竞平台的创业公司，在微信上进行了一个面试，我通过了。他们的老板很是想我去他们公司上班，恨不得马上就去他们公司上班。但是，鉴于我对这种创业公司发自内心的谨慎，所以，最后还是拒掉了。我要的6K，最后老板开到1W。我的内心是坚决的，所以，开再多我也不回去。我还是有私心的，其实，我就是觉得，我好久没面试了，为了面试看看自己啥水平，面试会出什么问题的目的随便投的。希望那位老板息怒，哈哈哈哈。

关于在家

我很赞成下面评论的内容。其实我需要坦白的是，我有一些肆无忌惮，态度不好。

关于总结

之前看到一个博主，每周都会写一篇总结，写点生活上的事情，写点工作上的，笔调不急不躁，描述着着走工作的的得失、心得以及总结等等。相当于一个周报吧。很是羡慕，决定试试能不能拾起来做一做。这就当做第一篇吧。届时，会开一个新的系列和tags来进行记录和分类属性。

还有就是，再看了这位博主的博客之后，我决定会在不就得将来对博客进行重构（先挖一个坑）。哈哈哈哈~~~

最后

今天就写这么多，无论未来怎么样，我都接受。都是我自己的选择的道路。我会尽自己最大的努力去挽回那些做错的事情。但愿我能够更加“成熟”（？？？）
